# 블로그 카테고리 가이드

- **Tech Deep Dive**: 기술/방법론 심층 분석, 적용 사례, 문제 해결 과정 공유.
- **Project Showcase**: 개인 프로젝트 기획→배포 여정, 기술 선택 이유, 교훈.
- **Tech Pulse**: 최신 트렌드, 컨퍼런스 후기, 주목할 오픈소스/도구 탐색.
- **Dev Hacks**: 생산성 팁, 환경 설정, 최적화/툴 사용법.
- **Troubleshooter's Corner**: 자주 만나는 에러/이슈 해결법과 체크리스트.
- **Retrospectives**: 프로젝트/경험 회고, 배운 점과 다음 계획.
